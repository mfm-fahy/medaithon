## üéØ Key Information

### System Architecture
- Frontend: Next.js on port 3000
- Backend: Express.js on port 5000
- Database: MongoDB
- Real-time: WebSocket

### Main Features
- Patient signup with QR code
- Unique patient ID generation
- Hospital navigation
- Real-time wait time updates
- 8 hospital departments

### API Endpoints
- POST /api/auth/signup
- POST /api/auth/login
- POST /api/visits
- GET /api/visits/navigation/:patientId

### WebSocket Events
- register, ping (client ‚Üí server)
- registered, wait-time-update, navigation-update (server ‚Üí client)
