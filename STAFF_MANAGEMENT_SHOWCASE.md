## 🎯 User Flow

```
Admin Dashboard
    ↓
Click "Add New Staff Member"
    ↓
Select Role (Doctor/Nurse/Pharmacist/Lab Tech/Biomedical)
    ↓
Fill Basic Information
    ↓
Fill Role-Specific Fields
    ↓
Click "Add [Role]"
    ↓
Form Validation
    ↓
Success Message
    ↓
Redirect to Dashboard
```

---

## 📊 Data Flow

```
Frontend Form
    ↓
Validation
    ↓
API Request (/api/auth/signup)
    ↓
Backend Processing
    ↓
Create User Record
    ↓
Create Staff Record
    ↓
Return Success
    ↓
Frontend Success Message
    ↓
Redirect to Dashboard
```
