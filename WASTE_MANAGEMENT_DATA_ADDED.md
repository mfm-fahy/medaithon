### **Data Models**
```
WasteRecord:
├─ wasteId (unique)
├─ date
├─ items (array with colorCode)
├─ totalQuantity
├─ department
├─ collectedBy
└─ status

CollectionTask:
├─ taskId (unique)
├─ date
├─ timeSlot (Morning/Evening)
├─ assignedStaff
├─ location
├─ status
├─ completedAt
└─ notes

DispatchSchedule:
├─ dispatchId (unique)
├─ weekStartDate
├─ weekEndDate
├─ dispatchDate
├─ processingPlant
├─ plantLocation
├─ totalWasteQuantity
├─ wasteBreakdown (yellow, red, blue, white)
├─ status
├─ dispatchedBy
├─ dispatchedAt
└─ notes
```
